前言：作为一名新手上周在学姐的帮助下成功搭建博客，今天分享一下我搭建博客的过程。

相关步骤

第一步：安装Node.js和配置Node.js环境

进入Node.js官网并下载最新版本并将其安装好之后可以检测一下PATH环境是否配置了Node.js。

安装好的标志：

打开命令提示符（cmd)，输入命令node -v和npm -v后如图所示

第二步：安装Git和配置Git环境
进入Git官网，下载最新版本并安装。
安装好的标志：在电脑任意位置单击鼠标右键，后又俩个选项：Git Bash Here和Git GUI Here

第三步：Github账户的注册和新建项目
注意：账户名.github.io，并且需要勾选Initialize this repository with a README

第四步：安装Hexo，在自己认为合适的地方创个文件夹。然后通过命令行进入到该文件夹里面

输入npm install hexo -g，开始安装Hexo

输入hexo -v，检查hexo是否安装成功

输入hexo init，初始化该文件夹（过程漫长，耐心等待）

输入npm install，安装所需要的组件，整个过程完成以后可以输入hexo g，首次体验Hexo

输入hexo s，开启服务器，访问该网址，正式体验Hexo

第五步:将Hexo与Github page联系起来，设置Git的username 和email

首先配置Deployment，在其文件夹中，找到_config.yml文件，修改repo 值（repo值是你在github项目里的ssh），然后安装一个扩展

在一个文件夹里面点击鼠标右键，点击Git Bash Here
1.输入cd ~/.ssh，查看是否有.ssh的文件夹
2.输入ls，列出该文件下的内容
3.输入ssh-keygen -t rsa -C “”（引号内输入个人邮箱），连续三个回车，生成密钥，最后得到了两个文件：id_rsa和id_rsa.pub
4.输入eval “$(ssh-agent -s)”，添加密钥到ssh-agent
5.再输入ssh-add ~/.ssh/id_rsa，添加生成的SSH key到ssh-agent
6.登录Github，点击头像下的settings，点击”SSH and GPG keys”
7.然后点击”New SSH key”新建SSH keys(title 可以随便写)，将id_rsa.pub文件里的内容复制在key框里（注意是id_rsa.pub文件）
8.输入ssh -T git@github.com，测试添加ssh是否成功
成功的标志：看到Hi后面是你的用户名

第六步：部署到Github
使用命令：hexo d -g 部署成功后便可访问地址：http://用户名.github.io 即可访问自己的博客
